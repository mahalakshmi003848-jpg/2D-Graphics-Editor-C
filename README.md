# 2D Graphics Editor in C

## Project Description

This project is a simple 2D graphics editor developed using the C programming language.  
The editor uses a 2D array of characters to create and display shapes in the terminal using '*' characters.

## Features

- Draw Rectangle
- Draw Line
- Draw Triangle
- Draw Circle
- Display Canvas
- Delete Shapes
- Modify Shapes

## Technologies Used

- C Language
- Functions
- Structures
- 2D Character Arrays

## Files in Project

- main.c
- canvas.c
- canvas.h
- shapes.c
- shapes.h

## How to Compile

Use the following command:

gcc main.c canvas.c shapes.c -o editor

## How to Run

Use the following command:

./editor

## Menu Options

1. Draw Rectangle
2. Draw Line
3. Draw Triangle
4. Draw Circle
5. Display Canvas
6. Delete Shape
7. Modify Shape
8. Exit
